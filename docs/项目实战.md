# 项目实战

meefood(我食)是一个基于物联网概念下的智能收货运营管理系统

![image-20240808151231606](https://shepi-1308499968.cos.ap-chengdu.myqcloud.com/img/image-20240808151231606.png)

## 角色与功能

* 管理员：查看订单，统计报表
* 运维人员：投放设备，拆除设备，维修设备
* 运营人员：补货
* 合作商：提供点位，坐收渔翁之利
* 消费者：在小程序下单商品
